# Entra P2 Feature Evidence

## Overview
This folder contains reference screenshots of Microsoft Entra ID P2 features that were available before the tenant's P2 licensing expiration on May 6, 2026.

The purpose of preserving these screenshots is to retain evidence of advanced identity governance and identity security capabilities for future Stage 3 (`AZ-500`) and later security-focused projects.

## Scope
These screenshots were captured as reference evidence only. They do not represent a completed standalone project, but rather preserved feature visibility from the Microsoft Entra admin center before the license expiration date.

## Preserved Feature Areas
- Privileged Identity Management (PIM)
- Identity Governance
- Identity Protection
