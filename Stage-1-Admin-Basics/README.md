# Stage 1: Admin and Identity Foundations

This stage focuses on the operational building blocks behind Azure administration. It aligns primarily with `AZ-104`, but it is organized publicly by project instead of by week so the work is easier to review and discuss.

## Objectives
- build confidence in Azure resource administration
- strengthen portal, CLI, and PowerShell workflows
- understand identity, RBAC, networking, storage, compute, and monitoring
- create portfolio-ready admin projects instead of isolated labs

## Focus Areas
- Microsoft Entra ID basics
- users, groups, and RBAC
- Azure virtual machines
- virtual networks, subnets, and NSGs
- storage and monitoring fundamentals
- governance and access control
