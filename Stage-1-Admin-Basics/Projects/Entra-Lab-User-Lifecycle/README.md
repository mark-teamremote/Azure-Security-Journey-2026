# Entra Lab User Lifecycle

## Overview
This project contains simple helper scripts used to create and clean up test users in Microsoft Entra ID for hands-on tenant practice.

## Purpose
This project supports early-stage Azure administration work by making it easier to repeat identity labs without manually creating and deleting users one by one.

## Files
- [Create-TestUsers.ps1](./Create-TestUsers.ps1)
- [Cleanup-TestUsers.ps1](./Cleanup-TestUsers.ps1)

## What It Demonstrates
- Microsoft Graph-based user administration
- repeatable test-user setup and cleanup
- practical identity operations aligned with `AZ-104`

## Notes
This project is intentionally simpler than the `Azure Identity Automation Lab`. Its purpose is lab setup and cleanup rather than portfolio-grade onboarding logic.
