# AZ-104 Mapping

This folder keeps the certification mapping for `AZ-104`, while the actual public work for Stage 1 is organized by project.

## Why project-first
A project-first layout is easier for recruiters, clients, and interviewers to understand because it shows outcomes instead of study chronology.
